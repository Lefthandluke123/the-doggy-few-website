Liedjeslijst (Liedjes-pagina)

Upload het JSON-bestand van je liedjeslijst (geëxporteerd uit je akkoorden-viewer), of plak de JSON hieronder. De site maakt er automatisch klikbare liedjes met transponeerbare akkoorden van. Dit verschijnt onder de H1/H2/beschrijving van de Liedjes-sectie.

Formaat: { "name": "...", "songs": [ { "title": "...", "content": "..." } ] }
